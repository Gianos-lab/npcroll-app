# Contributing to NPCRoll

Thank you for your interest in NPCRoll!

This project is currently developed and maintained by a single creator.  
For this reason, **external code contributions (PRs) are not being accepted**.

If you'd like to help improve NPCRoll, the best way is to share your thoughts:

### Feedback & Suggestions

## Why contributions are closed

- NPCRoll includes internal generation tools and content pipelines that are not part of the public repo.  
- The structure of the project is still evolving.  
- Keeping external PRs closed ensures consistency, security, and maintainability.

As the project grows, contribution guidelines may open in the future.
